## Pinout
- `RS = PORTA0`
- `RW = PORTA1`
- `E  = PORTA2`
- `D0 = PORTC0`
- `D1 = PORTC1`
- `D2 = PORTC2`
- `D3 = PORTC3`
- `D4 = PORTC4`
- `D5 = PORTC5`
- `D6 = PORTC6`
- `D7 = PORTC7`
